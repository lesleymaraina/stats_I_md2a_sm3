Workflow for Describing Continuous Data

  • describing continuous data before modeling involves a simialr sequence of steps as described for discrete values:

    1. define the observational unit
    2. identify the variable as continuous
    3. determine summaries of center and spread
    4. evaluate distribution shape across the measurement scale
    5. select visualizations that preserve numerical structure and distribution form

  • numerical summaries and graphical displays with how quantitative values are recorded and help describe what the measurements represent across patients

Workflow and Descriptive Interpretation

Each step in the workflow defines a component of continuous measurements described by statistical summaries or plots:

  • observational unit: determines what one recorded value represents within the study design
  • continuous variable: defines the measurement as a quantitative value on a numerical scale
  • summary: quantifies central tendency and dispersion
  • distribution analysis: evaluates symmetry, skewness, tails, and concentration across the scale
  • probability representation: expresses how much of the sample falls within domain-specific ranges
  • visualization: encodes the distribution in a form that preserves scale, shape, and comparability

Define the Observational Unit

Review:

  • observational unit is the entity represented by one row of data
  • each observational unit contributes one numerical value measured on a quantitative scale

Continuous examples of observational units include:

  • one patient encounter
  • one blood draw
  • one biopsy specimen
  • one laboratory assay result
  • one follow-up visit

Identify Continuous Variables

After the observational unit is established, continuous variables are recognized by how values are recorded:

  • values are quantitative rather than labels
  • values can vary across a numerical scale
  • values may take many distinct magnitudes within an interval

Examples in biomedical data include:

  • cytokine concentration
  • blood pressure
  • serum creatinine
  • tumor size

Descriptive summaries for continuous data help determine:

  • where values are centered
  • how widely values vary
  • how values are distributed across the scale

Determine the Data Summary for Continuous Variables

  • continuous variables are commonly described with summaries of center, spread, and shape :

    • mean : arithmetic center of the sample
    • median : middle ordered value
    • standard deviation : variability around the sample mean
    • interquartile range : spread of the middle 50% of observations
    • minimum and maximum : observed range endpoints
    • quantiles : positional summaries along the ordered scale

  • no single summary fully describes a continuous distribution

  • valid description combines numerical summaries with plots that show how values occupy the measurement scale

Continuous Distributions Describe Measurement Structure

  • continuous distributions describe how measured values are organized across the numerical scale

  • descriptive statistical analysis involves summarizing how values are distributed across observational units without removing the scale on which the measurements were recorded

For continuous variables, descriptive summaries show:

  • center → where values tend to cluster along the scale

  • spread → how widely values vary around the center

  • shape → whether the distribution is symmetric, skewed, heavy-tailed, or multi-peaked

  • range and tails → whether observations extend into unusually low or high values

Probability Quantifies Where Continuous Values Fall

For continuous data, descriptive probability is often expressed over an interval of values rather than at one exact value

For a sample of size \(n\):

\[ \hat{P}(a \le X \le b) = \frac{\text{number of observations with values between } a \text{ and } b}{n} \]

Formula defines the probability that a measured value falls within the interval from \(a\) to \(b\)

Where:

  • \(X\) : continuous variable
  • \(a\) : lower bound of the interval
  • \(b\) : upper bound of the interval
  • \(n\) : total number of observations

Visualizations for Continuous Data

  • plots are chosen to preserve the numerical scale and reveal how values occupy that scale

Visualizations for continuous variables include:

  • histogram : shows how observations accumulate across intervals of the scale
  • density plot : shows a smooth estimate of distribution shape
  • boxplot : summarizes median, quartiles, and tail extension

Mean and Median: Two Summaries of Center

Mean is expressed as:

\[ \bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i \]

Average of all observed values in the sample

Where:

  • \(\bar{x}\) : sample mean
  • \(n\) : number of observations
  • \(x_i\) : observed value for the \(i\)th observational unit
  • \(\sum_{i=1}^{n} x_i\) : total of all observed values

Median : middle value after the sample is ordered from smallest to largest

  • mean uses the magnitude of every value
  • median uses the order of values in the sample

Standard Deviation and Interquartile Range

Standard deviation:

  • measure that captures variability around the mean
  • summary of dispersion that quantifies how far observations extend from the mean across the entire distribution

\[ s = \sqrt{ \frac{1}{n - 1} \sum (x_i - \bar{x})^2 } \]

Where:

  • \(s\) : sample standard deviation
  • \(x_i\) : observed value for the \(i\)th observation
  • \(\bar{x}\) : sample mean
  • \((x_i-\bar{x})^2\) : squared deviation from the mean
  • \(n-1\) : denominator used for the sample-based estimate of variability

Standard Deviation and Interquartile Range

Interquartile Range:

  • width of the central interval containing the middle 50% of ordered observations
  • measure of dispersion
  • summary statistic for skewed variables and data with outliers

\[ IQR = Q_3 - Q_1 \]

Where:

  • \(Q_1\) : 25th percentile [value below which 25% of observations fall]
  • \(Q_3\) : 75th percentile [value below which 75% of observations fall]

IL-6: Histogram to Probability

  • continuous biomarker such as IL-6 can be treated as a random variable:

\[X=\text{IL-6 concentration [pg/mL]}\]

probability helps answer:

  • how likely are different IL-6 values?
  • where is probability concentrated?
  • how does probability change across the measurement scale?

IL-6: Histogram to Probability

  • histogram approximates how probability is distributed across intervals of IL-6

    • bar height reflects relative frequency, which approximates probability
    • dividing counts by total sample size converts counts into approximate probabilities

Interpretation

  • regions with taller bars correspond to higher probability density
  • low bars in the right tail indicate low-probability
  • total area across all bars corresponds to probability ≈ 1

\[\text{∑relative frequencies≈1}\]

IL-6: Density as a Probability Model

  • density curve represents an estimate of the probability density function [PDF] of IL-6

\[\text{f(x)=density at IL-6 value x}\]

Key properties:

  • \(f(x)≥0\)
  • total area under the curve equals 1

\[ \int_{-\infty}^{\infty} f(x)\,dx = 1 \] Interpretation:

  • height of the curve reflects relative likelihood of IL-6 values near x
  • area under the curve over an interval gives probability

Example:

\[ P(10 \le X \le 20) = \int_{10}^{20} f(x)\,dx \] - value corresponds to the region where most IL-6 values are concentrated or the probability that X falls in the 10 to 20 interval

IL-6: Boxplot and Probability

  • boxplot summarizes probability using quantiles

Quantiles partition probability:

  • \(Q_1\) : 25th percentile → 25% of observations fall below this value
  • median: 50th percentile
  • \(Q_3\) : 75th percentile

\[ P(X \le Q_1) = 0.25, \quad P(X \le \text{Median}) = 0.5 \]

Interpretation for IL-6:

  • middle 50% of probability lies between:

\[7.26≤X≤19.5\] - interval defines the highest-density region of the distribution

Outliers:

  • represent observations in low-probability regions
  • often correspond to clinically extreme cases

Summary

descriptive features define how the sample should be understood before later modeling decisions are made

plots and probabilities describing continuous variables help answer:

  • mean and median: describe where the sample is centered
  • standard deviation and IQR: describe how widely values vary
  • histograms and density plots: describe shape and concentration across the scale
  • boxplots: describe ordered spread and tail extension
  • interval probabilities: describe the fraction of values occupying clinically meaningful ranges

metrics help determine whether values are:

  • approximately symmetric
  • right-skewed or left-skewed
  • tightly concentrated or broadly dispersed
  • characterized by extended tails or unusually large values
